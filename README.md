## ðŸš€ Algorithms Included

### **1. Linear Regression**

* Predicts continuous values
* Covers gradient descent + closed form solution
* Includes visualization of regression line

---

### **2. Logistic Regression**

* Classification algorithm (binary)
* Sigmoid function + decision boundary
* Supports accuracy, confusion matrix, loss curve

---

### **3. Naive Bayes**

* Probabilistic classifier
* Works well for text classification
* Includes Gaussian Naive Bayes implementation

---

### **4. Support Vector Machine (SVM)**

* Classifier with maximum-margin decision boundary
* Kernels implemented where applicable
* Visualization for 2D datasets

---

### **5. Random Forest**

* Ensemble learning algorithm
* Multiple decision trees + majority vote
* Shows feature importance

---

### **6. K-Means Clustering**

* Popular unsupervised algorithm
* K initialization, centroid update, inertia calculation
* Includes visualization of clusters
